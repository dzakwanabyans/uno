This project can be used to count items otomatically using an ir proximity sensor, equipped with manual correction buttons (+)(-) and a reset button. System will be detected any object in front of sensor then save count results for display lcd. As long as the object is stationary in front of the sensor, the count will remain one. 
This tool is suitable to help calculate the amount of production, the number of people, the number of vehicles, etc.


1. Arduino Uno R3
2. LCD i2c 2x163
3. Jumper wire male female (conditional)
4. Power supply or battery 2x18650 for arduino
5. Buzzer
6. Tactile push button
